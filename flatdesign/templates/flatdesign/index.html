Flat Design Baseado no tema de bootstrap Flat UI

Flat design é um gênero minimalista de design de Interface de Usuário que é usado atualmente em interfaces gráficas para aplicativos e sites.

O estilo visa criar uma imagem 2D (ou plana, por isso o nome “flat”), não usando nenhum elemento estilístico adicional, como sombras ou gradientes. Uma interface de usuário de design plano parece simplificada, eficiente e fácil de ler.

O flat design é oposto ao esqueumorfismo, cujo método visa tentar mostrar representações próximas à realidade nos elementos visuais. Esse gênero de design opta por identificar seus aplicativos e objetos através de representações simples.

Além de atualmente estar em alta, o flat design se destaca pelas suas principais características:

Simplicidade

Seu foco é no desenvolvimento de layouts efetivos e interfaces limpas, evitando o uso de 3D, sombras e animações excessivas. Vale ressaltar que um design simples não representa uma criação simples, muito menos fácil.

Minimalista

Segundo o conceito de “menos é mais” o fator minimalista é utilizado para evitar confusão e desorganização no layout. O foco principal é na apresentação do conteúdo.

Cores

Quando se utilizam elementos simples para definir a estrutura do layout, a utilização de cores se torna fundamental para tornar o design vibrante e apelativo. As cores provocam emoções, e a teoria das cores deve ser considerada pelo designer.

Tipografia

Sem a utilização de textura, efeitos 3D, sombras e gradientes, a tipografia se torna de grande importância no flat design. A escolha da fonte é fundamental para trabalhar a mensagem do layout.

Design limpo

O design é limpo e vai direto ao ponto, exibindo detalhes e informações precisas. São os elementos exibidos da forma mais real, sem competições com sombras, efeitos, gradientes e afins.

Agilidade

O flat design é rápido para obter a atenção do usuário e levá-lo até o destino especificado pelo site. O foco são nos elementos essenciais, instruções claras e diretas. O flat design é objetivo e faz o trabalho de uma forma eficiente.
